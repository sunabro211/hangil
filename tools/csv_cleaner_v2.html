📊 사용 방법:
1. "클립보드에 복사" 클릭
2. Google Sheets 새 시트 열기
3. 이름을 "원본_한국어"로 변경 (시트 하단 탭 더블클릭)
4. A1 셀 클릭 → Ctrl+V로 붙여넣기
5. 1,000행이 자동으로 들어감
6. 그 다음 [TAB 2: 시트 자동 생성기]로 이동

이 Apps Script가 22개 언어 시트를 자동 생성합니다. 한 번 실행하면 신할라, 타밀, 네팔 등 22개 시트가 자동으로 만들어지고 GOOGLETRANSLATE 공식까지 자동 입력됩니다.

🤖 사용 방법:
1. Google Sheets 열기 → 메뉴 확장 프로그램 → Apps Script
2. 빈 화면이 나오면 기존 코드 모두 삭제
3. 아래 스크립트 복사 → 붙여넣기
4. Ctrl+S 저장
5. 상단 "실행" 버튼 클릭 (함수: setupAllLanguageSheets)
6. 권한 요청 → "허용" → 5분 기다림
7. 완료 후 시트로 돌아가면 22개 언어 시트가 자동 생성됨

전체 작업 흐름

1단계: 원본_한국어 시트 만들기 (5분)
[TAB 1]에서 클립보드 복사 → Google Sheets에 붙여넣기
2단계: 22개 언어 시트 자동 생성 (10분)
[TAB 2] Apps Script 실행 → 22개 시트 자동 생성, GOOGLETRANSLATE 공식 자동 입력
3단계: 자동 번역 완료 대기 (5-30분)
22개 언어 × 1,000문항 = 자동 번역. 시트가 처리할 때까지 기다림
4단계: 번역을 값으로 고정 (각 시트마다)
번역 칼럼 전체 선택 → 복사 → 같은 자리에 "값으로 붙여넣기" (Ctrl+Shift+V) → 통역사 수정 시 다시 영어로 안 돌아감
5단계: 권한 잠금 + 추적 시스템 (다음 단계)
통역사별 자기 시트만 편집 가능, 활동 자동 추적 (Apps Script 추가 예정)
⚠️ 중요한 알림:
- GOOGLETRANSLATE는 한도가 있어요. 22개 언어 × 1,000문항 한 번에 못 하면 시트가 멈출 수 있음.
- Apps Script가 그래서 50문항씩 끊어서 처리하도록 설계됐습니다.
- 만약 멈추면 그 시점에 멈춘 시트만 다시 실행하면 됩니다.
- 자동 번역 품질은 통역사 검수 시작점입니다 (50-70% 다시 쓰일 수 있음).
